# 🚗 Electric Vehicle Data Analysis Dashboard

## 📌 Objective
Analyzed Electric Vehicle (EV) trends including BEV and PHEV distribution 
across states to understand adoption patterns, model year trends, 
and state-wise market growth.

## 🛠️ Tools Used
- Tableau (Interactive Dashboard)
- Microsoft Excel (Data Preparation)

## 📊 Key Metrics Visualized
- Total vehicles by type (BEV vs PHEV)
- Model year trends
- State-wise distribution via maps
- Line charts and bar charts for trend analysis
